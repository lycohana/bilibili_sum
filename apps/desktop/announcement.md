# v1.16.0 更新说明

## 全新：VLM 理解型图文笔记

这是一版**从零重构**的图文笔记生成方式。和之前的"插图笔记"（在原文里插几张截图）完全不同——

- **画面驱动写作**：VLM 会阅读原始笔记和每一张截图的客观信息，然后以画面为线索重新组织一篇全新的笔记。段落结构、表达方式都会被重新编排，不再是原文+图片点缀。
- **精选配图**：从所有截图中只挑选最重要的 3-6 张，每张图紧跟对应段落，像专业教材一样讲一个知识点配一张图。
- **文字深度整合**：画面中的公式、图表、代码、界面等客观事实会被转化为自己的语言融入行文，而非"画面呈现了...""该图展示了..."这样的流水账描述。

在设置页 → 图文笔记形式中选择「VLM 理解型图文笔记」即可启用。

## 视觉 VLM 支持 Anthropic

设置页新增 Anthropic 作为视觉模型提供商。选择后图片格式、请求端点、响应解析自动适配，Claude 模型可直接用于画面理解和图文笔记合成。

## ASR 转写增强

- **多模态 ASR**：支持 OpenAI 兼容的多模态音频模型（如 mimo-v2-omni）进行语音转文字，长音频自动切片 + 重试。
- **SiliconFlow ASR 长音频**：突破 60 分钟限制，自动切片 + 并发识别，1 小时以上的视频也能正常转写。
- **切片参数可调**：切片时长和并发数在设置页可直接调整。

## 其他

- 修复 ASR 尾段时间戳溢出
