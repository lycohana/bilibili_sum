# v1.15.1 更新说明

这次主要补了一轮安全、登录和模型接入体验。

- Web 访问增加密钥保护，新安装会默认生成安全访问密钥。
- 设置页做了整理，已保存的 API Key 测试时不会被空值覆盖。
- 新增 B 站扫码获取 Cookies，遇到登录态或风控问题时处理更方便。
- 支持 macOS 桌面应用构建，并优化了 macOS 端窗口和打包体验。
- 修复运行环境同步可能覆盖 GPU 渠道 Python 的问题。
- 修复 Anthropic / Claude API 返回内容读取失败的问题。

如果更新后 Web 端提示需要访问密钥，请在桌面端或服务配置中查看当前密钥后再登录。
