# v1.16.0 更新说明

这次重点打磨了**图文笔记（VLM 理解型）**的生成质量，并扩展了 ASR 转写和视觉模型的支持范围。

## 图文笔记质量重构

- **提示词全面重写**：从"插图笔记"（原文+图片点缀）重构为"理解型笔记"（以画面为线索重新组织文章）。VLM 会主动调整段落结构、合并重复观点、用精炼语言重新表达。
- **杜绝流水账句式**：禁止"画面呈现""该画面""上图""这张图展示"等句式。画面中的知识点直接融入行文，像专业教材一样自然。
- **数据源头优化**：帧描述从叙述性文本改为客观事实列表（`key_facts`），避免 VLM 看到"该画面展示了..."的输入后模仿输出。
- **图片分散插入**：强制要求段落→图片→段落的交替结构，只精选最重要的 3-6 张图，禁止图片堆积在文章末尾。
- **合成稳定性**：预过滤低质量帧（importance<2.5 不送入 VLM），合成超时从 120s 提高到 300s，大视频不易超时回退。

## 视觉 VLM 提供商

- **新增 Anthropic 支持**：设置页下拉可选 Anthropic 作为视觉模型提供商，图片格式自动转换（OpenAI ↔ Anthropic）、响应自动解析。
- **第三方兼容端点智能回退**：SiliconFlow 等 Anthropic 兼容端点的图片 API 会自动使用 OpenAI 格式，避免不兼容导致的超时或失败。

## ASR 转写增强

- **多模态 ASR 集成**：支持 OpenAI 兼容的多模态音频 API（如 mimo-v2-omni），音频自动切片（3 分钟/段）+ 5 次重试，空响应自动回退 `reasoning_content`。
- **SiliconFlow ASR 长音频支持**：突破单次 60 分钟限制，长音频自动 ffmpeg 切片 + 并发识别（ThreadPoolExecutor），大幅缩短转写时间。
- **切片与并发可配置**：多模态 ASR 和 SiliconFlow ASR 的切片时长和并发数均可在设置页调整。

## 其他优化

- 图文笔记进度条移除，界面更简洁
- 设置页搜索索引补全（图文笔记 User Template）
- 分支 `auto_generate_mindmap` 默认值调整
- 思维导图 JSON 解析、图文笔记预设/重跑链路修复
- 设置页导航栏吸顶适配
